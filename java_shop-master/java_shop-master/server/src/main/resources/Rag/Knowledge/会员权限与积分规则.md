# 会员权益与积分规则
## 1. 会员等级
- 普通会员：注册即得，享基础购物折扣9.8折
- 白银会员：累计消费满1000元升级，享9.5折+生日礼券
- 黄金会员：累计消费满5000元升级，享9折+专属客服+免运费
- 钻石会员：累计消费满20000元升级，享8.5折+优先发货+年度定制礼品

## 2. 积分获取
- 购物消费：每消费1元累计1积分
- 评价晒单：完成订单后评价+晒图，可获得50-100积分
- 签到：每日签到可获得5-10积分，连续7天额外奖励30积分

## 3. 积分使用
- 积分可兑换优惠券、实物礼品或抵扣订单金额（100积分=1元）
- 积分有效期为1年，过期自动清零
