# 首页产品列表与标签栏重构设计

## Context

对商城首页的产品列表区域（pc-thing-list）和热门标签栏（tag-sidebar）进行 Apple Store 风格的现代化重构，提升视觉层次和交互体验。

---

## 设计方案：Apple Store 风格

### 1. 产品卡片重构

**尺寸与圆角：**
- 图片区域高度：120-220px → 180-260px
- 卡片圆角：8px → 16px

**阴影与边框：**
- 默认阴影：`0 4px 20px rgba(70, 132, 226, 0.08)`
- Hover 阴影：`0 12px 32px rgba(70, 132, 226, 0.15)`
- 无边框，依靠阴影区分

**Hover 效果：**
- transform: translateY(-6px)
- 阴影加深
- 图片轻微放大(1.05x)

**快速操作按钮：**
- 从圆形 `+` 改为胶囊形状"加入购物车"
- 背景透明，hover 时滑入蓝色背景

---

### 2. 热门标签重构

**形状：**
- 胶囊形（高度 28px，圆角 14px）

**样式：**
- 默认：白色背景 + 蓝色边框 + 蓝色文字
- Hover：渐变背景(`linear-gradient(135deg, @primary-blue, @primary-blue-hover)`) + 白色文字 + 光晕阴影
- 选中：渐变背景 + 白色文字 + 微光晕阴影

---

### 3. KingKong 分类导航重构

**图标：**
- 尺寸：48px → 56px
- 圆形背景带蓝白渐变

**布局：**
- 增加间距，分类之间更透气

**交互：**
- Hover：向上弹跳(translateY -4px) + 轻微放大(1.08x) + 阴影加深
- 选中：蓝色渐变背景，图标变白，文字加粗

---

## 验证方式

1. 启动前端开发服务器：`npm run dev`
2. 访问首页，验证：
   - 产品卡片更大、hover 有明显的向上浮动效果
   - 热门标签 hover 时有渐变背景变化
   - KingKong 图标 hover 时有弹跳效果
3. 检查各状态（默认、hover、选中）是否符合设计
